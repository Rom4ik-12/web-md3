# MD3 Web Elements (Materializing the Browser)

Этот проект — моя попытка частично **материализовать браузер**, а также воссоздать экспериментальную концепцию **Material Design 3 Expressive (MD3 Expressive)** в веб-среде, поскольку сама компания Google так и не реализовала полноценную Expressive-спецификацию для веба.

Проект включает в себя интерактивную витрину кастомных веб-элементов в стиле MD3 (включая уникальный морфинг-лоадер и видеодемонстрации их работы прямо на странице), набор пользовательских стилей (UserStyles) для популярных сайтов (YouTube, Gemini, ChatGPT) и стили для кастомизации интерфейса Firefox (`userChrome.css`).

---

## 🌟 Основные особенности проекта

### 🌀 1. MD3 Morphing Spinner (Морфинг-Лоадер)
*   **Что это:** Кастомный лоадер, который плавно перетекает между 6 абстрактными органическими формами.
*   **Как это сделано:** Набор ключевых кадров SVG, преобразованных с помощью фильтра слияния (`feGaussianBlur` + `feColorMatrix` для gooey-эффекта) и запеченных в бинарные маски CSS.
*   **Где используется:** В интерактивном превью, а также встроен прямо в темы для YouTube, Gemini, ChatGPT и Firefox, заменяя стандартные скучные индикаторы загрузки.

### 🌊 2. Wavy Progress Bars (Волнистые индикаторы прогресса)
*   **Линейный индикатор:** Бегущая «змейка» по скругленной траектории (`MD3 Wavy Linear`).
*   **Круговой индикатор:** Волнистый прерывистый круг с бесконечной плавной анимацией вращения.
*   **Применение:** Заменяет стандартный плоский красный прогресс-бар в плеере YouTube на невероятный волнистый индикатор!

### 🎨 3. UserStyles (Темы для Stylus)
Полная визуальная переработка интерфейсов популярных веб-сервисов под гайдлайны Material Design 3 Expressive с интеграцией видеодемонстраций прямо на витрине:
*   **YouTube MD3 (`youtube-md3.user.css`):**
    *   Кастомный волнистый прогресс-бар (MD3 Wavy Linear) в видеоплеере.
    *   Замена стандартных индикаторов загрузки на анимацию морфинга (Morphing Spinner) практически на всех элементах интерфейса.

*   **Главная фишка:** Во время загрузки страницы заменяет стандартную иконку сайта (favicon) на вкладке на кастомный анимированный морфинг-лоадер.

## 📂 Структура репозитория

*   `index.html` — Интерактивная витрина со всеми элементами, видеодемонстрациями и стилями.
*   `chatgpt-md3.user.css` — Тема для ChatGPT.
*   `gemini-md3.user.css` — Тема для Google Gemini.
*   `youtube-md3.user.css` — Тема для YouTube.
*   `userChrome.css` — Стили для кастомизации Firefox.
*   `md3_youtube.mp4` / `md3_gemini.mp4` / `md3_gpt.mp4` / `md3_firefox.mp4` — Оптимизированные видеодемонстрации работы стилей в высоком качестве.

## 🛠 Инструкция по установке

### 1. Как открыть витрину элементов (index.html)
Просто откройте файл `index.html` в любом браузере или запустите локальный сервер. На странице представлены интерактивные демонстрации всех элементов, видеопревью работы тем и готовые куски кода (HTML/CSS) для их копирования.

### 2. Как установить темы для сайтов (UserStyles)
1.  Установите расширение **Stylus** для вашего браузера ([Chrome Web Store](https://chrome.google.com/webstore/detail/stylus/clngdbkbkbpnknglcjiaocjbopechmke) / [Firefox Add-ons](https://addons.mozilla.org/ru/firefox/addon/styl-us/)).
2.  Откройте Stylus в браузере, нажмите **«Создать стиль» (Write new style)**.
3.  Скопируйте содержимое нужного файла (например, `youtube-md3.user.css`) и вставьте в текстовое поле Stylus.
4.  Нажмите кнопку **«Сохранить» (Save)** в левом верхнем углу. Тема применится автоматически при посещении сайта.

### 3. Как настроить интерфейс Firefox (userChrome.css)
1.  В адресной строке Firefox введите `about:config` и примите риск.
2.  Найдите параметр `toolkit.legacyUserProfileCustomizations.stylesheets` и переведите его в `true`.
3.  Перейдите в папку вашего профиля Firefox (ее путь можно найти на странице `about:support` в графе «Папка профиля»).
4.  В папке профиля создайте подпапку с именем `chrome`.
5.  Поместите файл `userChrome.css` из этого репозитория внутрь созданной папки `chrome`.
6.  Полностью перезапустите Firefox. Интерфейс браузера обновится!
